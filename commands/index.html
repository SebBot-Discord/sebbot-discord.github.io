COMMANDS

  NOTE: Commands with an asterisk ("*") in front of them are deprecated and may not work properly


Moderation Commands

Command Description
s!ban <user> <reason> Bans the mentioned <user> from the server
s!kick <user> <reason> Kicks the mentioned <user> from the server
